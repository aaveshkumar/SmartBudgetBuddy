2. Database Schema (MySQL)
-- Users Table
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    email VARCHAR(100) NOT NULL UNIQUE,
    password VARCHAR(255) NOT NULL,
    type ENUM('jobseeker','employer','admin') DEFAULT 'jobseeker',
    verified TINYINT(1) DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- Categories Table
CREATE TABLE categories (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    slug VARCHAR(150) NOT NULL UNIQUE
);

-- Technologies Table
CREATE TABLE technologies (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(50) NOT NULL
);

-- Education Levels Table
CREATE TABLE education_levels (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(50) NOT NULL
);

-- Jobs Table
CREATE TABLE jobs (
    id INT AUTO_INCREMENT PRIMARY KEY,
    title VARCHAR(255) NOT NULL,
    description TEXT NOT NULL,
    category_id INT NOT NULL,
    employer_id INT NOT NULL,
    location VARCHAR(100) NOT NULL,
    salary VARCHAR(50),
    type ENUM('Full-time','Part-time','Contract','Internship') DEFAULT 'Full-time',
    status ENUM('Pending','Approved','Rejected') DEFAULT 'Pending',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (category_id) REFERENCES categories(id),
    FOREIGN KEY (employer_id) REFERENCES users(id)
);

-- Applications Table
CREATE TABLE applications (
    id INT AUTO_INCREMENT PRIMARY KEY,
    job_id INT NOT NULL,
    user_id INT NOT NULL,
    resume_path VARCHAR(255) NOT NULL,
    applied_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (job_id) REFERENCES jobs(id),
    FOREIGN KEY (user_id) REFERENCES users(id)
);

-- Saved Jobs Table
CREATE TABLE saved_jobs (
    user_id INT NOT NULL,
    job_id INT NOT NULL,
    saved_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    PRIMARY KEY(user_id, job_id),
    FOREIGN KEY (user_id) REFERENCES users(id),
    FOREIGN KEY (job_id) REFERENCES jobs(id)
);

-- Blog Table (SEO Content)
CREATE TABLE blog (
    id INT AUTO_INCREMENT PRIMARY KEY,
    title VARCHAR(255) NOT NULL,
    content TEXT NOT NULL,
    slug VARCHAR(255) NOT NULL UNIQUE,
    meta_title VARCHAR(255),
    meta_description VARCHAR(160),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- SEO Logs Table
CREATE TABLE seo_logs (
    id INT AUTO_INCREMENT PRIMARY KEY,
    url VARCHAR(255) NOT NULL,
    meta_score INT DEFAULT 0,
    last_updated TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

4. Security & Anti-Spam Configuration

Use reCAPTCHA v2 or v3: store site key and secret in /config/env.php.

Implement CSRF tokens: generate per form, store in session.

Validate all inputs: htmlspecialchars(strip_tags($input)).

Resume file validation: only .pdf, .docx, .txt. Max 5MB.

Login throttling: block after 5 failed attempts per IP for 15 mins.

Session fingerprint: combine $_SERVER['REMOTE_ADDR'] + $_SERVER['HTTP_USER_AGENT'].

5. SEO & Meta Configuration

Clean URLs via .htaccess rewrite rules:
Example: /jobs/software-developer-delhi → job.php?id=123

Dynamic meta tags per job and blog:

<title><?= $job['title'].' | '.$job['location'].' | '.$company_name ?></title>
<meta name="description" content="<?= substr(strip_tags($job['description']),0,155) ?>">


Schema.org JSON-LD for each job.

Auto sitemap generation: /cron/sitemap.php.

Auto ping to search engines: /cron/ping_search_engines.php.

sitemap.php	Daily	Rebuild sitemap.xml
ping_search_engines.php	Daily	Notify Google/Bing
daily_backup.php	Daily	Backup DB + uploads
expired_jobs_cleanup.php	Weekly	Delete or archive expired jobs
daily_jobs_digest.php	Daily	Send email digest to users
image_optimize.php	On Upload	Compress uploaded images

On Replit, schedule using external free cron service like cron-job.org.
